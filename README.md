📌 Project Details

Family Card Management System is a web-based application developed using PHP and MySQL to manage family information, members, assets, gifts, and monthly financial records. The system creates a digital family database where administrators and users can easily store, update, and manage family-related data.

The main goal of this project is to provide an organized and secure platform for managing family profiles and financial information. It allows users to track monthly income and expenses, manage family member records, and maintain gift or asset information efficiently.

This system is designed with a role-based access control mechanism, allowing different users such as Admin, Editor, and User to perform specific tasks according to their permissions.

🎯 Objectives

Store and manage family information digitally

Maintain records of family members and assets

Track monthly income and expenses

Provide role-based access control

Enable fast searching and reporting of data

⚙️ Technologies Used

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: PHP

Database: MySQL

Libraries: Chart.js, FontAwesome

Server: Apache (XAMPP)

🧩 Key Features

👨‍👩‍👧‍👦 Family Profile Management

📊 Monthly Income and Expense Tracking

🎁 Gift / Product Record Management

🔍 Search and Pagination System

📷 Image Upload with Modal Zoom

📈 Graphical Reports using Chart.js

👥 Role-Based Access (Admin / Editor / User)

📱 Responsive User Interface

🚀 Future Improvements
